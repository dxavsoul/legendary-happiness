## 🚀 Getting Started

Ready to launch your career? Let's go!

1. **Clone the repository:**
   ```bash
   git clone https://github.com/dxavsoul/legendary-happiness.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd legendary-happiness
   ```

3. **Install dependencies:**
   ```bash
   npm install
   # or yarn install / pnpm install / bun install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   # or yarn dev / pnpm dev / bun dev
   ```

Open [http://localhost:3000](http://localhost:3000) in your browser — and start building the profile that gets you hired! 🎉
